# 快餐员工培训游戏

一款基于 Phaser 3 + TypeScript 开发的横版 2D 教学小游戏，帮助快餐行业新员工快速掌握不同品牌的餐品组装流程。

## 游戏特色

- 🍔 **三关对应三大品牌**：麦当劳经典牛堡线、肯德基炸鸡堡线、华莱士卷饼线
- 🎯 **订单式训练**：屏幕上方滚动显示当前订单，玩家按正确顺序拖放食材
- ⚡ **传送带玩法**：食材从传送带上源源不断出现，训练反应速度
- 💯 **品质评分系统**：错误顺序扣品质分，连续三次错误触发重做
- 🎵 **动态 BGM**：剩余时间低于 30 秒时 BGM 节奏加快
- 📊 **详细统计**：关末展示平均组装秒数与最易错步骤排行
- ⭐ **星级评价**：根据品质分获得 1-3 星评价
- 💾 **本地存档**：自动保存最高通关星级记录
- 📱 **触控支持**：鼠标和触屏均可操作

## 游戏玩法

1. **选择关卡**：在主菜单选择想要训练的品牌线
2. **查看订单**：屏幕上方 ticker 滚动显示当前订单名称和步骤数
3. **拖放食材**：从下方传送带上拖拽食材到右侧包装区
4. **按序组装**：按照订单要求的正确顺序依次放入食材
5. **完成订单**：所有步骤正确完成后进入下一单
6. **通关条件**：完成 10 单即可过关

## 关卡 JSON 字段说明

关卡配置文件位于 `src/data/levels/` 目录下，每个品牌对应一个 JSON 文件。

### Level 根字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `id` | string | 关卡唯一标识 | `"mcdonalds"` |
| `name` | string | 关卡显示名称 | `"麦当劳经典牛堡线"` |
| `brand` | string | 品牌标识 | `"mcdonalds"` |
| `primaryColor` | string | 品牌主色（十六进制） | `"#FFC72C"` |
| `secondaryColor` | string | 品牌副色（十六进制） | `"#DA291C"` |
| `bgColor` | string | 背景色（十六进制） | `"#27251F"` |
| `totalOrders` | number | 通关所需订单数 | `10` |
| `timeLimit` | number | 关卡时间限制（秒） | `300` |
| `items` | Item[] | 可用食材列表 | - |
| `distractors` | Item[] | 干扰项食材列表 | - |
| `orders` | Order[] | 订单模板列表 | - |

### Item 字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `id` | string | 食材唯一标识 | `"bun_top"` |
| `name` | string | 食材显示名称 | `"面包顶"` |
| `color` | string | 食材颜色（十六进制） | `"#F4D03F"` |
| `shape` | string | 食材形状 | `"half-circle"` |

**支持的形状类型：**
- `circle` - 圆形
- `square` - 方形
- `half-circle` - 半圆形
- `rectangle` - 矩形
- `leaf` - 叶子形
- `slice` - 切片形
- `ring` - 环形
- `splash` - 溅洒形
- `strip` - 条状
- `irregular` - 不规则形

### Order 字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `id` | string | 订单唯一标识 | `"double_cheese"` |
| `name` | string | 订单显示名称 | `"双层吉士堡"` |
| `steps` | string[] | 组装步骤（食材 ID 数组，按顺序） | `["bun_bottom", "sauce", "patty", ...]` |
| `exclusions` | string[] | 排除的食材 ID 列表 | `["pickle"]` |

### 配置示例

```json
{
  "id": "mcdonalds",
  "name": "麦当劳经典牛堡线",
  "brand": "mcdonalds",
  "primaryColor": "#FFC72C",
  "secondaryColor": "#DA291C",
  "bgColor": "#27251F",
  "totalOrders": 10,
  "timeLimit": 300,
  "items": [
    {
      "id": "bun_top",
      "name": "面包顶",
      "color": "#F4D03F",
      "shape": "half-circle"
    }
  ],
  "distractors": [
    {
      "id": "fish_patty",
      "name": "鱼排",
      "color": "#85C1E9",
      "shape": "rectangle"
    }
  ],
  "orders": [
    {
      "id": "double_cheese_no_pickle",
      "name": "双层吉士堡 不要酸黄瓜",
      "steps": ["bun_bottom", "sauce", "patty", "cheese", "patty", "cheese", "onion", "bun_top"],
      "exclusions": ["pickle"]
    }
  ]
}
```

## 评分规则

### 星级评定

| 品质分 | 星级 |
|--------|------|
| ≥ 90 | ⭐⭐⭐ |
| ≥ 70 | ⭐⭐ |
| ≥ 50 | ⭐ |
| < 50 | 无星 |

### 得分计算

```
总得分 = 订单完成数 × 100 + 品质分 + 剩余时间奖励
剩余时间奖励 = floor(剩余时间 / 10) × 5
```

## 技术栈

- **游戏引擎**：Phaser 3
- **开发语言**：TypeScript
- **构建工具**：Vite
- **样式方案**：Canvas 2D 绘制（色块风格）
- **数据存储**：localStorage
- **容器化**：Docker + Nginx
